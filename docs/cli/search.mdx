Free-text search across the Rocksky catalog.

## Usage

```bash
rocksky search [options] <query>
```

## Parameters

| Name      | Required | Description                                         |
| --------- | -------- | --------------------------------------------------- |
| `<query>` | Yes      | Search string (artist, album, track, or account)    |

## Example

```bash
rocksky search "linkin park"
```
