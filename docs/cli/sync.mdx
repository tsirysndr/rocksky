---
title: "sync"
description: "Sync local Rocksky data from the AT Protocol."
icon: "rotate"
---

Pull your scrobbles and metadata from the AT Protocol into local state.
Useful when running the [local scrobble API](/cli/scrobble-api) or after
restoring from a backup.

## Usage

```bash
rocksky sync
```

Run `rocksky help sync` for the optional flags.
